# KaleidoSwap TypeScript SDK

The official TypeScript SDK for KaleidoSwap - a decentralized exchange for Bitcoin and RGB assets.

## Overview

KaleidoSwap SDK provides a comprehensive TypeScript interface for interacting with the KaleidoSwap API. This SDK enables seamless integration of onchain and Lightning Network swaps for RGB assets.

## Key Features

- 📊 **Asset Management** - Utilities for managing trading pairs and assets
